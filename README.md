## 왜 만들었나요?
개발을 할 때 항상 디자인이 많은 리소스를 잡아먹었습니다.
'padding은 몇으로 해야하지?', '컬러는 어떤걸 써야할까?'
이런 고민들을 줄일 수 있게, 최소한의 노력으로 더 좋은 디자인을 가진 어플리케이션을 개발할 수 있도록
여러 서비스들을 분석해 디자인 가이드를 만들었습니다.

## 이런 것들을 알 수 있어요!
- Typography
- Color
- Layout
- Card
- Arrange
